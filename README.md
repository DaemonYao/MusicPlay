# 													:sailboat:	模仿网易云音乐网站项目

## :ear_of_rice:  1.运行方法：

​		:alien: 将项目 clone 到本地，使用 npm run serve 运行。

​		:baby:需要将后端接口也同时拷贝到本地，将后端API和项目同时运行才能呈现效果。

​		:cactus:后端API网址：https://github.com/Binaryify/NeteaseCloudMusicApi。
